# Photos:

## 1. My decision
The first thing that I want to self host is a method of photo storage as my family takes a lot of photos I want to be able to store them all and be able to easily access them and look through them. After looking at a few opting I settles with Immich. This is a piece of self hosted software that can be used to mange both photo and videos. I picked this one as it looks similar to google photos which is what people in my family are used to using. I also used this one as it has a very good searching feature using AI to recognise people and objects making it easy to search through. It is also easy to share photos with other people.

## 2. Installing Immich
In order to install Immich I followed the instructions on the website to get the docker compose file. https://docs.immich.app/install/docker-compose/. I had a few issues when setting it up with the HDD not being mounted so I had to setup the software a few times however once it was running properly it was very easy to run through the setup process to get to the main page.
